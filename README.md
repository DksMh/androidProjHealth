# Second_Project
- 안드로이드 프로젝트
- 수료과정 두번째 프로젝트입니다.
- 제가 담당한 파트는 운동입니다.

### 운동 파트 설명
1. 운동화면으로 들어올 때 유저의 키와 체중 정보받기

2. 운동 메인화면 
   - 유저 정보와 표준 체중, BMI정보 보여주기
   
3. 운동 시작화면
  - 시작버튼을 누르면 현재 위치에서 지도에 선이 그어지기
  - 운동을 하는 도중, 중지버튼을 누르면 시간이 중지되고 계산되어 칼로리 계산하기

4. 기록 화면
  - 날짜마다 걸은 거리와 칼로리 누적하여 유저에게 보여주기
